## Set a target before entering

Choose a category, maximum spend, and minimum useful discount. This prevents a clearance hunt from becoming random shopping.

## Run the route

1. Check the store's dedicated clearance area.
2. Check endcaps in the target department.
3. Look for shelf labels with recent date changes.
4. Verify the exact item and variant with an official scanner or price checker.
5. Inspect condition, missing parts, and return eligibility.

## Trust the checkout price—not the sticker color

A clearance-colored label can be old, store-specific, or attached to the wrong item. Match the barcode and confirm the live price.

## Record a useful find

```text
Store / area:
Exact product:
UPC or model:
Verified price:
Condition:

```

This is the information another community member can actually use.

## The buy-or-pass test

Buy when the item solves a real need, the exact variant is correct, the price is verified, and the return policy is acceptable. Pass when scarcity is the main reason you want it.
